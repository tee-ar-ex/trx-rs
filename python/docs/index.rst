trxrs
=====

`trxrs` is the Python interface to `trx-rs`, a Rust implementation of the TRX
tractography file format.

The package is designed for Python users who want fast TRX reads with
memory-mapped payload access, while keeping behavior close enough to
`trx-python` for direct parity testing.
